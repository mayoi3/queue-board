# 順番待ちボード / Queue Board

VRChatのイベント等で、参加者の順番を表形式で管理できるワールド用ギミックです。
参加者は「列に並ぶ／列から抜ける」で登録・離脱、主催は各行のチェック（✅）で進行状況を切り替えます。
表示は10件×ページで最大100名。
退出後も名簿は残り、再参加は末尾から。お誕生日会のツーショット列、受付・点呼などに使えます。
